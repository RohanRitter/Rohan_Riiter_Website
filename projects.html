Projects

Dinesty

A personal website I created to rate our favorite restaurants and wine estates around South Africa. Users can add their favorite foods and drinks for easy access worldwide. Built using HTML, CSS, and JavaScript.

CES Spinning Cube Website

A website created for a company where, as the user scrolls or swipes, a cube spins and unfolds to display the page content. Developed with HTML, CSS, and JavaScript.

GIS

A GIS map project for a company I worked for. The map allows users to draw polygons, store them in a database, and retrieve them later for continued work. Additional features include changing the basemap, using Street View, loading GeoJSON files, and more. Built with HTML, CSS, JavaScript, and PostgreSQL. (GeoServer is currently offline due to private hosting fees).

Mini Games
